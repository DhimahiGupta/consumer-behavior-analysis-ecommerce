# Consumer Behavior Analysis in E-commerce

Overview
Analysis of online consumer behavior using survey data.

Objective
To identify factors influencing purchase decisions.

Dataset
- Primary survey (Likert scale)

Tools & Techniques
- R
- Logistic Regression

Key Results
- Personalization significantly increases purchase probability
- Pricing has lower impact

Insights
- Customer engagement > pricing strategy
